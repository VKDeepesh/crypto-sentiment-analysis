# Crypto Market Sentiment Analysis

## Objective
This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and cryptocurrency trader performance using historical trading data.

## Dataset
- Bitcoin Fear & Greed Index
- Historical Trader Data (Hyperliquid)

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Analysis Performed
- Number of trades by sentiment
- Total Closed PnL by sentiment
- Average Closed PnL by sentiment
- Buy vs Sell performance
- Top profitable coins
- Top profitable traders
- Average trade size by sentiment
- Average trading fees by sentiment
- Winning vs Losing trades
- Correlation between Fear & Greed Index and Closed PnL

## Conclusion
The analysis indicates that market sentiment affects trading activity and profitability. Traders tend to trade more actively during Greed periods, while Fear periods require more cautious risk management.
